# 音声・マイク仕様

最終確認日: 2026-07-15

## 確定: お客様音声

- 登録済みMP3がある場合はMP3を再生する。
- MP3が再生できない場合でも会話進行を停止しない。
- 音声ファイルがない文章は、対応ブラウザーで音声合成を使用できる。
- 12カ月点検は従来音声を使用する。
- 複数時刻から早い時刻を選ぶ返答、説明の食い違い解消後、家族相談後の再連絡、近い店舗・家族来店への同意、汎用の会話継続には登録済みMP3を使用する。
- 車検誘致は四国めたんと剣崎雌雄を選択できる。
- 選択した車検誘致音声はLocalStorageへ保存する。

## 確定: マイク開始

- スタッフ発話から始まる車検誘致は、「ロープレ開始」を押した直後に入力欄を消去して音声認識を開始する。
- お客様音声の再生終了後にスタッフ入力欄を消去する。
- 現在は終了から180ミリ秒後に音声認識開始を試みる。
- 音声自動再生をOFFにした場合も同じ流れで認識開始を試みる。

## 確定: 音声認識

- Web Speech APIを使用する。
- 言語は`ja-JP`。
- 中間結果と連続認識を有効にする。
- 最終認識結果の後、2秒待って文章の完結を判定する。
- 完結していれば自動送信する。
- 途中と判定した場合は「はい」と相槌を再生して認識を再開する。
- 認識の`end`発生後、必要な場合は250ミリ秒後に再開する。
- 音声認識非対応ブラウザーではテキスト入力を使用する。

## 確定: 現在の無音判定

音量や無音継続時間を測定する独立した無音検出はない。現在の終了判断はブラウザーの最終認識結果と文章完結判定に依存する。
