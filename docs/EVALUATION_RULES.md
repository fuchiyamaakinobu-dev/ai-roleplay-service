# 評価・採点ルール

最終確認日: 2026-07-15

## 確定: 共通

- 得点範囲は0〜100点。
- 達成項目は会話全体で一度でも認識されれば加点対象となる。
- 良かった点、改善点、判定ポイント、推奨トークを表示する。
- 合格・不合格の固定基準はまだ設けない。

## 確定: 12カ月点検

配点:

- 依頼を受け止める: 8点
- その他ご用命・気になる点: 15点
- 作業時間: 8点
- 引取希望理由: 13点
- 来店メリット: 13点
- 土日などの提案: 13点
- 他店舗などの提案: 8点
- 選択肢を残す: 8点
- 次の約束: 14点

お客様の理由に合わない提案項目は満点条件から除外し、該当する配点合計を100点換算する。仕事理由では他店舗提案を必須にせず、距離・運転不安では土日提案を必須にしない。

減点:

- 来店提案前に引取を確定: 25点
- 来店を強く迫る: 10点
- 引取を強く拒否: 10点
- お客様の事情に提案が合わない: 15点

## 確定: 車検誘致

17項目の配点合計は100点。

- 必要キーワードグループをすべて満たすと項目達成。
- 日時提案は「月・日・時」の具体値が必要。
- 予約復唱は、先に確定した月・日・時との一致が必要。
- 案内不足による聞き返し1回につき2点減点。
- 聞き返し減点は最大20点。
- 入庫日時確定後の任意項目は、省略しても聞き返し減点を付けない。
- 本人確認はお客様名を必要とし、「お客様でしょうか」だけでは達成にしない。
- 名乗りは店舗名と担当者名の両方を必要とする。
- 待ち方と車両状態の確認は、対象語だけでなく質問・確認の形を必要とする。
