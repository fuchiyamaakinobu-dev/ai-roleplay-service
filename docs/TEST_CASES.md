# テストケース

最終確認日: 2026-07-15

## 自動テストとして固定すべき基準

### 共通

- シナリオの配点合計が100点
- 表示文と音声登録文が一致
- `ready`音声ファイルが存在し、極端に小さくない
- 四国めたん・剣崎雌雄の車検誘致音声が存在

### 12カ月点検

- 追加整備未確認でも減点して会話継続
- 作業時間と追加整備確認を同じ発話で認識
- 「オイル交換もお願いします」と回答
- 再確認時は「そのほかは大丈夫です」と回答
- 作業時間を説明済みなら同じ質問を繰り返さない
- 引取理由が仕事なら土日・時間帯提案に反応
- 日付だけなら時間帯を聞き返す
- 日付と時間がそろうと予約確定
- 引取を即時確定すると減点

### 車検誘致

- 「もしもし」だけでは本人確認ステップを進めない
- 名乗りを先に行った場合は本人確認を聞き返す
- 本人確認と名乗りをまとめて話しても認識
- 名乗りと「いつもお世話になっております」をまとめて話すと、減点条件を保ったまま車検案内へ進む
- 日頃の利用へのお礼なしでも減点して進行
- 入庫日時未確定の終話で「いつ行けばいいんですか？」
- 入庫日時確定後は任意案内を省略可能
- 省略発話中に予約復唱があれば加点
- 模範17発話で100点・正常終話

## 手動ブラウザ確認

- お客様音声が途中で切れない
- 音声終了後にマイクが開始する
- スタッフ入力欄が前回発話のまま残らない
- 短い無音でロープレ自体が終了しない
- 発話候補が変化する
- 会話分岐が事情に合う
- 採点が実際の応対と一致する
- A4縦・白黒で会話と評価が印刷される
- 管理画面の下書きが公開画面へ即時反映されない
- 「公開する」後に公開画面へ反映される
