# お客様発話パターン

最終確認日: 2026-07-15

## 確定: 管理場所

- 会話候補と分岐: `scenario.js`
- 音声ID、音声文、MP3: `audio-db.js`
- Firebase管理画面から公開された文章: Firestore `roleplay/public`

文章を変更する場合は、表示文、音声登録文、MP3内容の一致を確認する。

## 確定: 12カ月点検の主な候補群

- 作業時間の質問
- オイル交換希望
- その他はないという回答
- 引取依頼
- 仕事による来店困難
- 距離・運転不安
- 他店比較
- 説明の食い違い
- 家族相談
- 来店可能性を示す回答
- 日付・時間の確認
- 予約確定

同じ候補群では未使用候補を優先し、全候補使用後に再利用する。

## 確定: 車検誘致

各ステップに以下を持つ。

- 条件を満たした場合のお客様回答
- 条件不足時の聞き返し
- 対応する音声ID

## 確定: 使用停止中

`audio-db.js`上で以下は`disabled`となっており、通常候補から外れている。

- `customer_pickup_05.mp3`
- `customer_objection_competitor_02.mp3`
